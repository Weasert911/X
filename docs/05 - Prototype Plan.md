# Prototype Plan (MVP)

## Goal
Build a playable version that proves:
- Chaos is fun
- Multiplayer works
- Core loop is engaging

---

## Phase 1: Core Setup

- Player movement
- Basic map (gali layout)
- Camera (top-down angled)

---

## Phase 2: Core Mechanics

### Chaos System
- Basic meter
- 2 levels (Low, High)

### Jugaad Items
- 1 throwable (chappal)
- 1 explosive (cooker)
- 1 vehicle (rickshaw)

---

## Phase 3: Multiplayer

- 4–8 players
- Basic sync (movement + actions)

---

## Phase 4: Events

- Police spawn
- Basic enemy AI

---

## Phase 5: Signals (Basic)

- 3 signals
- Radial menu

---

## Playtest Checklist

- Do players laugh?
- Do players troll each other?
- Does chaos feel fair?

---

## Success Criteria

- Matches feel different each time
- Players create their own fun
- No major confusion during chaos

---

## Do NOT Build Yet

- Cosmetics
- Progression
- Store/monetization
- Advanced UI

---

## Next Step After MVP

Expand:
- Items
- Chaos events
- Maps
